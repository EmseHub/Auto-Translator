# Auto-Translator

**Auto-Translator** ist eine Browser-Erweiterung für Chrome und Firefox, die Microsoft Teams-Chatnachrichten per Klick über DeepL übersetzt und die Übersetzung direkt ins Textfeld zurückschreibt.

## Funktionen
- Nachricht in Teams eingeben und per Klick übersetzen lassen  
- DeepL wird automatisch geöffnet und genutzt  
- Übersetzung wird ins Teams-Textfeld eingefügt  

## Unterstützte Browser
- Google Chrome/Edge
- Mozilla Firefox  

## Installation
1. Lade den Projektordner herunter oder klone das Repository.  
2. Öffne die Erweiterungsverwaltung deines Browsers.  
3. Lade die Erweiterung über die `manifest.json`.
